## Features
  - Target multiple companies
  - Tailor the printed copy
  - Write the template only once
  - Link back to projects and contributions
  - Component-based design
  - Scalable CSS with styled-components
  - Comprehendible code

## Install

```sh
npm install
```

## Usage

```sh
npm start
```

or

```sh
npm run build
```
